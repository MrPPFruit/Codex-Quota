# Codex Quota v0.2.0-preview.2

这是 Windows 11 x64 的第二个公开预览版，重点修正 Windows 气泡与 macOS 版本之间明显的材质、尺寸和排版差异；macOS 版本不受影响。

## Windows 视觉重构

- 收起态与展开态保持为 52×52 DIP 和 130×78 DIP，可见表面不再额外内缩。
- Windows 11 22621 及以上优先使用系统 Desktop Acrylic，并用动态窗口区域形成圆形或连续圆角外形。
- 内部彩色色场、额度语义色、字体层级和两行布局按 macOS 版本重新对齐；色场改为固定几何中的连续二维柔光，不再出现扇区色带或随 hover 追踪缩放。
- 删除旧版黑色实底、独立彩色描边和外发光；系统材质不可用时使用明确的不透明中性浅色降级。
- 5 小时不可用时完整回退到“本周”；两组额度均不可用时显示“额度 / --”。

## 下载与校验

下载：

- `Codex-Quota-v0.2.0-preview.2-windows-x64.zip`
- `Codex-Quota-v0.2.0-preview.2-windows-x64.zip.sha256`

本 Preview 没有 Authenticode 商业代码签名，SmartScreen 可能显示未知发布者。SHA-256 只能确认文件与本 Release 资产一致，不能替代开发者身份认证。请只从 `MrPPFruit/Codex-Quota` 官方 Release 下载，完整解压到固定目录后运行；不要直接从 ZIP 内运行。

## 预览边界

Windows runner 会验证编译、测试、固定 WPF 结构截图、x64 PE、self-contained single-file、未签名状态、ZIP 内容与 SHA-256。固定 PNG 不包含 DWM 桌面合成，因此 Desktop Acrylic 背景采样、圆形边缘抗锯齿、100%–200% DPI、Reduce Motion、UI Automation、拖动/hover 和空闲动画 CPU 仍需要真实 Windows 11 实机验收。

安装、升级和完整验收步骤见 [Windows 分发说明](https://github.com/MrPPFruit/Codex-Quota/blob/main/docs/windows-distribution.md)。
