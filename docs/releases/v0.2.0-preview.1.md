# Codex Quota v0.2.0-preview.1

这是首个 Windows 11 x64 公开预览版，同时保留现有 macOS 版本。Windows 版使用原生 WPF/.NET 10，完整解压后只需运行 `CodexQuota.exe`，不需要另装 .NET。

## Windows 新增

- Codex 启动时显示额度气泡，Codex 退出时自动隐藏。
- 读取 5 小时与周剩余额度；5 小时不可用时回退显示周额度。
- 52×52 DIP 收起态、悬停展开、拖动和位置记忆。
- 无任务栏、无激活 Topmost 窗口及托盘控制。
- 默认注册当前用户登录启动项，可在托盘中关闭。
- 仅接受官方 Codex MSIX package identity 和同签名 helper；不从 PATH 执行同名程序。

## 下载与校验

下载：

- `Codex-Quota-v0.2.0-preview.1-windows-x64.zip`
- `Codex-Quota-v0.2.0-preview.1-windows-x64.zip.sha256`

本 Windows Preview 没有 Authenticode 商业代码签名，SmartScreen 可能显示未知发布者。SHA-256 只能确认文件与本 Release 资产一致，不能替代开发者身份认证。请只从 `MrPPFruit/Codex-Quota` 官方 Release 下载，完整解压到固定目录后运行；不要直接从 ZIP 内运行。

## 预览边界

CI 已验证 C# 核心测试、Windows x64 PE、self-contained single-file、未签名状态、ZIP 内容和 SHA-256。由于发布前没有 Windows 实机，以下项目需要下载后完成首轮验收：Store Codex helper 路径、真实额度、SmartScreen、窗口焦点、hover/拖动、DPI、登录启动和退出后的子进程残留。因此本版本明确标记为 Prerelease，不是 Windows 稳定版。

安装、升级和验收步骤见 [Windows 分发说明](https://github.com/MrPPFruit/Codex-Quota/blob/main/docs/windows-distribution.md)。
