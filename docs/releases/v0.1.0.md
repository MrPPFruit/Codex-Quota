# Codex Quota v0.1.0

这是 Codex Quota 唯一的公开预览 Release。它在同一个发布页提供 macOS 与 Windows 的当前安装包；平台适配和缺陷修复会替换对应资产，不会在没有新增用户功能时增加新的公开版本号。

## 资产

- `Codex-Quota-macos-arm64.zip`：Apple silicon Mac，macOS 26 或更高版本。
- `Codex-Quota-macos-arm64.zip.sha256`：macOS ZIP 的 SHA-256 校验文件。
- `Codex-Quota-windows-x64.zip`：Windows 11 x64，要求 Microsoft Store 安装的官方 ChatGPT 桌面应用（包含 Codex）。
- `Codex-Quota-windows-x64.zip.sha256`：Windows ZIP 的 SHA-256 校验文件。

## 功能

- 显示 Codex 5 小时与周额度及重置时间；5 小时窗口不可用时自动显示周额度。
- 以独立、可拖动的桌面气泡显示；悬停展开完整额度信息。
- 在 Codex/官方桌面宿主运行时显示，宿主退出时隐藏；不抢占当前应用焦点。
- macOS 通过登录项、Windows 通过当前用户登录启动项保持后台伴随。

## 安全与安装边界

这仍是 **Prerelease**：macOS ZIP 为 ad-hoc 签名且未经 Apple 公证；Windows ZIP 尚无 Authenticode 商业代码签名。请只从本仓库官方 Release 下载并先校验 SHA-256。校验值只能确认文件与发布资产一致，不能替代开发者身份或平台恶意软件检查。

- macOS：解压后将应用移至 `/Applications`；首次被 Gatekeeper 阻止时，前往“系统设置 → 隐私与安全性”选择“仍要打开”。不要关闭 Gatekeeper 或执行全局移除隔离属性的命令。
- Windows：完整解压 ZIP 到固定目录后运行 `CodexQuota.exe`；首次可能出现 SmartScreen，确认来源和校验值后可选择“更多信息 → 仍要运行”。不要直接从 ZIP 内运行。

详细说明见 [macOS 分发说明](../macos-distribution.md) 和 [Windows 分发说明](../windows-distribution.md)。
