# 更新记录

## 0.1.0 - 2026-07-13

- 唯一公开预览 Release，同时提供 Apple silicon macOS 与 Windows 11 x64 安装包及 SHA-256 校验文件。
- 显示 Codex 的 5 小时与周额度及重置时间。
- 支持独立悬浮、拖动与位置记忆、悬停展开和状态色流。
- Codex 启动时显示、退出时隐藏。
- 安装到 Applications 后首次启动会自动尝试注册为 macOS 登录项，用户可随时关闭。
- Windows 版提供 Codex 生命周期联动、托盘与用户级登录启动；Windows 包为 self-contained 单文件 ZIP，无需另行安装 .NET。
- macOS 使用 ad-hoc 签名且未公证，Windows 尚未进行 Authenticode 签名；统一 Release 保持 Prerelease 标记。
